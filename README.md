### Prerequisites
- Node.js
- npm or yarn

### Installation
1. Clone the repository
   ```bash
   git clone https://github.com/username/project-name.git
   ```
2. Install dependencies
   ```bash
   npm install
   # or
   yarn install
   ```
3. Run the development server
   ```bash
   npm run dev
   # or
   yarn dev
   ```
